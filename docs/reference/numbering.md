# Numbering Module

This module provides functions for numbering antibody sequences.

## Functions

### `number_ab_seq`
Number antibody sequences using standardized schemes.

### `align_ab_seqs`
Align numbered antibody sequences.

## Classes

### `NumberedAntibody`
Represents a numbered antibody sequence.

### `NumberedAntibodyWithGermline`
Represents a numbered antibody with germline assignment.

### `AntibodyAlignment`
Represents an alignment of antibody sequences.

### `Germlines`
Contains germline gene information.

### `NumberedScFv`
Represents a numbered scFv sequence.

### `number_scfv_seq`
Number scFv sequences.
