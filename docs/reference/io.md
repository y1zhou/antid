# I/O Module

This module provides I/O functions for sequences and structures.

## Functions

### `fasta2seq`
Read sequences from FASTA files.

### `struct2seq`
Extract sequences from structure files.

### `df2pdb`
Convert DataFrames to PDB format.

### `gemmi_convert`
Convert between structure formats.

### `struct2df`
Convert structures to DataFrames.

## Classes

### `RCSBDownloader`
Download structures from RCSB PDB.
