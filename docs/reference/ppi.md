# PPI Module

This module provides functions for analyzing protein-protein interactions.

## Functions

### `collect_ab_ag_contacts`
Find contacts between antibody and antigen.

### `collect_within_ab_contacts`
Find intra-antibody contacts.

### `get_contacts`
Get atomic contacts in a structure.

### `get_atomic_sasa`
Calculate solvent accessible surface area.

## Classes

### `LiabilityScanner`
Scan sequences for liabilities.
